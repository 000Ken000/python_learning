##install-安装python3和三方库
### 安装python3 
1. 官网下载python3.7安装包，mac终端输出``$ python3``,检查是否安装成功。
2. 安装pip.
3. 安装第三方库
```
$ pip3 install numpy
$ pip3 install pandas
$ pip3 install matplotlib
$ pip3 install scipy
$ pip3 install --upgrade pip
$ pip3 install tushare
$pip3 install tushare -upgrade 
```
